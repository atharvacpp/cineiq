# 🎬 CineIQ - Explainable Hybrid Movie Recommendation System

CineIQ is an advanced, hybrid movie recommendation engine that blends **Content-Based Filtering**, **Collaborative Filtering**, and **Sentiment Analysis** to provide highly accurate, explainable movie recommendations.

## 🌟 Project Overview

Traditional recommendation systems are "black boxes." CineIQ solves this by integrating an **Explainability Layer** powered by LIME (Local Interpretable Model-agnostic Explanations) and rule-based templates to tell users exactly *why* a movie was recommended.

### Key Features
- **Hybrid Engine**: 40% TF-IDF Content Similarity + 60% Surprise SVD Collaborative Filtering.
- **Sentiment Re-Ranking**: Analyzes movie reception using VADER / DistilBERT to boost critically acclaimed films.
- **Explainability**: Generates natural language explanations (e.g., *"Because you liked 'Toy Story', sharing genres like Animation, Comedy and actor(s) Tom Hanks."*)
- **Taste Profile**: Generates Plotly radar charts and visual affinities based on a user's watch history.
- **MLflow Tracking**: Complete tracking of hyperparameter sweeps, RMSE, Precision@5, and Recall@5.

## 🛠️ Setup Instructions

### 1. Environment Setup
```bash
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Dataset Requirements
CineIQ requires datasets placed in `data/raw/`:

| Dataset | Source | Files |
|---------|--------|-------|
| **MovieLens 25M** | [GroupLens](https://grouplens.org/datasets/movielens/25m/) | `movies.csv`, `ratings.csv` |
| **TMDB 5000** | [Kaggle](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata) | `tmdb_5000_movies.csv`, `tmdb_5000_credits.csv` |
| **IMDB 50K Reviews** | [Kaggle](https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews) | `imdb_reviews.csv` |

### 3. Run the Pipeline
We provide a master shell script to automate everything:
```bash
bash run.sh
```

This script will:
1. ✅ Install Python dependencies
2. ✅ Process CSVs into an optimized Parquet file
3. ✅ Train the SVD model and track metrics via MLflow
4. ✅ Run a quick verification test
5. ✅ Start the FastAPI backend at `http://localhost:8000`
6. ✅ Start the Streamlit Dashboard at `http://localhost:8501`

**On Windows:** Run `.\run.bat` instead.

## 🧠 How It Works

### Hybrid Recommendation Engine
1. **Content-Based (40%)**: TF-IDF vectorizes movie "soup" (genres + cast + keywords + overview + director), then computes cosine similarity.
2. **Collaborative (60%)**: SVD from Surprise library learns latent user-item factors from 25M ratings.
3. **Blending**: Scores are normalized to [0,1] and blended: `hybrid = 0.4 × content + 0.6 × CF`

### Sentiment Re-Ranking
- Uses VADER for fast sentiment scoring on movie overviews
- Blends with TMDB community vote average
- Formula: `final_score = hybrid_score × 0.7 + sentiment_score × 0.3`

### Explainability
When a recommendation is served:
1. The **Explainer** checks the source (content, CF, or hybrid).
2. For **Content-Based**: Tries LIME text perturbation first. If unavailable, uses rule-based shared feature extraction (matching genres, cast, directors between query and recommended movie).
3. For **Collaborative**: Adds "Users with similar taste loved this."
4. For **Sentiment**: Adds reception indicator (positive / mixed / negative).
